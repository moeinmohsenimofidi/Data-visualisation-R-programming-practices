The Old Faithful is a geyser in Yellowstone park, famous for erupting at pretty regular intervals. R comes with the faithful dataset containing information about the eruption durations (eruptions column) and the waiting time between eruptions (waiting column).

We can group measurements using the following snippet, which creates a new dataset, faithful_split, containing an additional column: group. This column contains just values "low" and "high", identifying two groups of measurements that depend on the waiting time between eruptions.

faithful_split <- faithful %>%
  mutate(group = if_else(waiting < 69, "low", "high"))
